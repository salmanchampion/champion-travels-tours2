
# CHAMPION TRAVELS - ERROR FREE BLOGGER THEME (v2)

### 1. INSTALLATION
1.  Go to **Blogger Dashboard** > **Theme**.
2.  Click the **Arrow (â–¼)** next to "Customize" > **Edit HTML**.
3.  Delete EVERYTHING and paste the content of `blogger_theme.xml`.
4.  Click **Save**.

### 2. HOW TO CUSTOMIZE (Using Layout)
Go to **Blogger Dashboard** > **Layout**. 

You now have full control without touching code:

*   **Header Logo:** Look for the widget titled **"Header Logo"**. It is now an **HTML/Javascript** widget to prevent upload errors.
    *   Click "Edit".
    *   In the "Content" box, replace the image URL `https://i.postimg.cc/PJS59Bqw/champion-logo-1.png` with your own logo URL.
    *   Click Save.
*   **Main Menu:** Click Edit on "Main Menu" to change navigation links.
*   **Hero Slider:** Click Edit on the "Hero Slider" widget. You will see a script at the bottom of the content box where you can change image URLs.
*   **Services:** There are 3 separate widgets for services. You can edit the text/icon for each.
*   **Hajj & Umrah Packages:** These are HTML widgets inside a Section.
    *   To Add a Package: Click "Add a Gadget" -> Select "HTML/Javascript" -> Paste the card HTML code (below).
    *   To Edit a Package: Click "Edit" on an existing package widget.
